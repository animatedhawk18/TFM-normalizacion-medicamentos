# Normalización semántica de medicamentos

Este proyecto implementa un sistema híbrido (reglas + machine learning)
para la estandarización de descripciones de medicamentos según la
Resolución 3311 de 2018 en Colombia.

## Contenido
- Extracción semántica
- Validación estructural
- Modelo de clasificación
